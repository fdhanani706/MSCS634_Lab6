# MSCS634_Lab6

## Purpose
This lab explores association rule mining using Apriori and FP-Growth algorithms to identify frequent itemsets and generate meaningful insights from transactional data.

## Key Insights
- Milk, bread, and butter frequently occur together.
- Strong association rules were identified using confidence and lift.
- FP-Growth performed faster than Apriori.

## Challenges
- Selecting appropriate support and confidence thresholds.
- Understanding differences between algorithms.

## Conclusion
Association rule mining helps uncover hidden patterns in transactional data, which can be used for business decision-making such as product placement and recommendations.
